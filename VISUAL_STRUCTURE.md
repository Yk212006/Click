## 🔌 Responsive Design

### Mobile (< 768px)
```
Single Column Layout
┌─────────────┐
│  Logo       │
│  [Menu]     │
│  [CTA]      │
├─────────────┤
│  Full Width │
│  Sections   │
│             │
│             │
├─────────────┤
│  Stacked    │
│  Grid Items │
│  (1 column) │
└─────────────┘
```

### Tablet (768px - 1024px)
```
Optimized Column Layout
┌──────────────────────┐
│ Logo   [Nav]   [CTA] │
├──────────────────────┤
│  Content             │
│                      │
│  2-Column Grid       │
├──────────────────────┤
│  Footer (2 columns)  │
└──────────────────────┘
```

### Desktop (> 1024px)
```
Full Layout
┌──────────────────────────────┐
│ Logo    [Nav Items]    [CTA]  │
├──────────────────────────────┤
│  Content (Full Width)         │
│  3-Column Grid                │
│  6-Column Complex Layout      │
├──────────────────────────────┤
│  Footer (4 columns)           │
└──────────────────────────────┘
```

Borders:          Light Gold/Gray
Hover States:     Gold text/border
